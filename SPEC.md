# Portfolio Website Specification

## 1. Project Overview

**Project Name:** Arjun Sharma Portfolio  
**Type:** Personal portfolio website  
**Core Functionality:** Showcase skills, projects, and background of a young developer/student  
**Target Users:** Recruiters, potential collaborators, educators

## 2. UI/UX Specification

### Layout Structure

- **Hero Section:** Full-height intro with name, tagline, animated background
- **About Section:** Bio and background
- **Skills Section:** Grid of skill cards with icons
- **Projects Section:** Project cards with descriptions
- **Footer:** Simple copyright

### Visual Design

**Theme:** Dark modern with gradient accents

**Color Palette:**
- Background: `#0a0a0f` (deep dark)
- Card Background: `#16161f`
- Primary Accent: `#6366f1` (indigo)
- Secondary Accent: `#22d3ee` (cyan)
- Tertiary Accent: `#f472b6` (pink)
- Text Primary: `#f8fafc`
- Text Secondary: `#94a3b8`

**Typography:**
- Headings: 'Outfit' (Google Fonts)
- Body: 'Outfit'
- Hero Name: 64px bold
- Section Titles: 32px

**Visual Effects:**
- Gradient mesh background
- Glassmorphism cards
- Hover animations on cards
- Smooth scroll behavior
- Staggered fade-in animations

### Components

**Hero:**
- Large name with gradient text
- Tagline
- Animated floating shapes background

**About Card:**
- Photo placeholder (geometric shape)
- Bio text

**Skill Cards:**
- Icon + title
- Hover lift effect

**Project Cards:**
- Project title
- Description
- Tech tags

## 3. Functionality Specification

### Sections

1. **Hero** - Name, tagline, scroll indicator
2. **About** - Full bio
3. **Skills** - Grid of skills organized by category
4. **Projects** - 3 project cards
5. **Footer** - Copyright

### Interactions

- Smooth scroll to sections
- Hover effects on all interactive elements
- Scroll-triggered animations
